White Paper 6: Perturbation and Functional Genomics (e.g., LINCS)
Causal Inference in Biological Systems
Abstract
Perturbation datasets measure how gene expression changes when the system is actively altered (e.g., drugs, gene knockdowns). These datasets enable causal understanding of biological networks.
1. Biological Basis
Perturbations include:
Chemical treatments
Genetic modifications
Environmental changes
2. Measurement
Gene expression is measured:
Before and after perturbation
3. Scientific Objective
Identify:
Cause-effect relationships
Regulatory dependencies
Pathway structure
4. Data Characteristics
High-dimensional
Condition-specific
Often sparse and noisy
5. Scientific Impact
Perturbation data enables:
Drug discovery
Target identification
Network reconstruction
Unifying Scientific Perspective
Across all datasets:
1. Cells as Dynamical Systems
Cells evolve over time under:
Internal regulation
External signals
2. Cells as Stochastic Systems
Biological processes are inherently noisy:
Molecular fluctuations
Environmental variability
3. Cells as Information-Processing Systems
Cells:
Receive signals
Process them through networks
Produce decisions (gene expression changes)
4. Core Scientific Challenge
Understanding:
How reliably cells encode information
How noise limits decision-making
How biological systems maintain function despite uncertainty
