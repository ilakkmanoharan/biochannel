# BioChannel promo videos

Generated by:

```bash
cd /path/to/bio-channel
.venv/bin/pip install -r private/marketing/requirements-video.txt
.venv/bin/python private/marketing/build_promo_video.py
```

**Requirements**

- `GOOGLE_API_KEY` in the repo-root `.env` (Gemini narration).
- macOS **`say`** for built-in voice-over (export `BIOCHANNEL_VIDEO_VOICE` to pick another voice; default `Samantha`).
- **`imageio-ffmpeg`** supplies an `ffmpeg` binary for muxing.

**Outputs**

`BioChannel_promo.mp4` is **not committed** (too large for GitHub); it appears locally after a successful build.

| File | Description |
|------|--------------|
| `BioChannel_promo.mp4` | Slideshow + narration (local only) |
| `biochannel_promo_narration.txt` | Spoken script from Gemini |
| `biochannel_promo_voice.wav` | Narration audio |

Place screenshots as PNG under `private/marketing/images/` before building.
