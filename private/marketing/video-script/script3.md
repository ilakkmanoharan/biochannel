# 🎤 **Founder Talk (30-Second Live Pitch)**

---

**[Opening — steady, direct]**

Cells make decisions under noise—
but today, we don’t have a way to measure how reliable those decisions are.

---

**[What you built — clear, fast]**

We’re building a system that models cells as **information-processing networks**.

Signals go in,
biological pathways process them,
and gene expression comes out.

---

**[What it does — concrete]**

Using real datasets:

* Single-cell RNA-seq shows **cell decisions as clusters**
* Time-series data measures **information loss**
* Drug response data predicts **sensitivity and resistance**

---

**[Why it matters — sharper tone]**

This lets us quantify how reliable biological signaling is,
understand variability across cells,
and predict how interventions change outcomes.

---

**[Closing — strong, slightly slower]**

We’re turning biology from observation…
into a **measurable, predictive system**.
